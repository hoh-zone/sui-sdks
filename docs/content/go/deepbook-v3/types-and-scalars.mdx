- `MainnetMarginPools` / `TestnetMarginPools`

## 精度换算规则

典型换算方式（见 `transactions/deepbook.go`）：

- 价格：`u64(price * FloatScalar * quoteScalar / baseScalar)`
- 数量：`u64(quantity * baseScalar)`
- 推荐在业务层保留 `float64` 用户单位，仅在构建交易时转换为链上单位。

`py-sdks` 是 `sui-sdks` 仓库下的 Python SDK 实现。

## Package layout

- `pysdks.sui`: 节点客户端、交易、签名、验证、多签、zkLogin
- `pysdks.bcs`: BCS 编码解码基础能力
- `pysdks.deepbook_v3`: DeepBook V3 客户端与交易构建
- `pysdks.walrus`: Walrus 客户端与文件/Blob 辅助
- `pysdks.seal`: Seal 阈值加解密与会话密钥

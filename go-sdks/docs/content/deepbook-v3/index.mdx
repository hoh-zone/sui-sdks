`deepbook_v3` 是 Go 版本的 DeepBook V3 SDK 绑定，提供：

- 客户端查询封装（基于 `sui_dryRunTransactionBlock`）
- 交易构建合约封装（Spot、Margin、Admin、Governance、Flash Loan）
- 主网/测试网默认配置（coins、pools、package IDs）
- 参数类型、精度换算、基础校验与错误模型

---
title: Sui Programmable Transaction Basics (Go)
---

```go
tx := transactions.NewTransaction()

coin := tx.SplitCoins(tx.Gas(), []transactions.Argument{
    tx.PureBytes([]byte{100}),
})

tx.TransferObjects([]transactions.Argument{coin}, tx.PureBytes([]byte("0xSomeSuiAddress")))
```

常见命令：

- `SplitCoins`
- `MergeCoins`
- `TransferObjects`
- `MoveCall`
- `Publish`
- `MakeMoveVec`

对象输入：

- `tx.Object("0x...")`
- `tx.Object(transactions.Inputs.ObjectRef(...))`
- `tx.Object(transactions.Inputs.SharedObjectRef(...))`
- `tx.Object(transactions.Inputs.ReceivingRef(...))`
